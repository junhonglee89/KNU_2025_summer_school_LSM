5. Upward shortwave radiation at the surface (\(LW_u\))#

\[ C_{soil} \frac{\partial T_{s}}{\partial t} = (1-\alpha) \cdot SW_d + LW_d - \textcolor{red}{LW_u} - H - \lambda E - G \tag{1.1} \]

  • Definition: The longwave radiation emiited from the land surface.

  • Equation:

\[ LW_u = \epsilon_s \sigma T_s^4 \tag{5.1} \]

where \(\epsilon_s\) is the emissivity of the land surface, \(\sigma\) is the Stefan-Boltzmann constant (5.67 \(\times\) 10-8 W/m2/K4)



Emissivity of the land surface (\(\epsilon_s\))#

Typically 0.90 ~ 0.99.

EMISSIVITY_S = 0.95


Stefan-Boltzmann constant (\(\sigma\))#

SB_CONSTANT = 5.67 * 1.0e-8 #W/m2/K4


Practice 5.1#

Compute \(LW_u\) for a given range of \(T_s\) (250-310 K).

import numpy as np
import xarray as xr
import seaborn as sns
import matplotlib.pyplot as plt
\[ LW_u = \epsilon_s \sigma T_s^4 \tag{5.1} \]
# Define Ts range
Ts = np.arange(250.0, 310.0+1, 1)

# Calculate LWu
def calculate_LWu(Ts):
    LWu = EMISSIVITY_S * SB_CONSTANT * Ts**4
    return LWu

LWu_np = calculate_LWu(Ts)

# Put LWd into xarray
LWu = xr.Dataset(
                 {'LWu': (['Ts'], LWu_np)},
                 coords={'Ts': Ts},
                 attrs={'units': 'W/m2',
                         'long_name': 'Upward longwave radiation'},
)

Practice 5.2#

Predict \(T_s\) in 2025 at KNU Natural Science Building 2. Assume that

  1. \(T_s\) at initial time is 290 K

  2. \(C_{soil}\) is \(c_{soil} \cdot d_s\), where \(c_{soil}\) is heat capacity of soil (2.0 \(\times\) 106 J/m3/K) and \(d_s\) is depth of soil (0.07 m)

  3. \(\alpha\) is 0.3

  4. \(LW_d\) is 400 W/m2

  5. \(H\) and \(\lambda E\) are 30% of \(SW_d\)

  6. \(G\) is 0 W/m2

→ Predict \(\textcolor{red}{T_s}\), \(\textcolor{red}{LW_u}\) by prescribing \(SW_d\), \(LW_d\), \(H\), \(\lambda E\), \(G\):

\[ C_{soil} \textcolor{red}{\frac{\partial T_{s}}{\partial t}} = (1-\alpha) \cdot SW_d + LW_d - \textcolor{red}{LW_u} - H - \lambda E - G \tag{1.1} \]


Rewrite

\[ C_{soil} \frac{\partial T_{s}}{\partial t} = (1-\alpha) \cdot SW_d + LW_d - LW_u \ - H - \lambda E - G \tag{1.1} \]

into

\[ C_{soil} \frac{\partial T_{s}}{\partial t} = (1-\alpha) \cdot SW_d + LW_d - \textcolor{red}{\epsilon_s \sigma T_s^4} - H - \lambda E - G \tag{5.2} \]

→ apply finite difference method (forward scheme),

\[ C_{soil} \textcolor{red}{\frac{T_s^{n+1} - T_s^{n}}{\Delta t}} = (1-\alpha) \cdot SW_d + LW_d - \epsilon_s \sigma T_s^4 - H - \lambda E - G \tag{5.3} \]

where \(T_s^{n+1}\) and \(T_s^{n}\) are \(T_s\) at \(n+1\)th and \(n\)th timestep.

And then rewrite Eq. 5.3,

\[ \textcolor{red}{T_s^{n+1}} = \textcolor{red}{T_s^{n} + \frac{\Delta t}{C_s}} \left[ (1-\alpha) \cdot SW_d + LW_d - \epsilon_s \sigma T_s^4 - H - \lambda E - G \right] \tag{5.4} \]
import function_package as fp
import pandas as pd

# Setup

dt      = 5                  # s
Ts_init = 295                # K
C_SOIL  = 2.0 * 1.0e6 * 0.07 # J/m2/K
albedo  = 0.3
LWd     = 400                # W/m2


datetime_UTC_2025 = pd.date_range(start="2025-01-01", end="2026-01-01", freq=f'{dt}s')
time_dimension_size = len(datetime_UTC_2025)


lat_degree = fp.LAT_KNU
lon_degree = fp.LON_KNU


SWd = fp.calculate_SWd(lat_degree, lon_degree, datetime_UTC_2025) # W/m2
SWd = SWd.values # convert xarray to numpy array for the speed


H   = 0.3 * SWd # W/m2
LE  = 0.3 * SWd # W/m2
G   = 0.0       # W/m2

# Predict Ts

### Create an array for Ts
Ts = np.zeros(time_dimension_size)

### Initial value of Ts
Ts[0] = Ts_init

### Solve finite difference equation
for n in np.arange(1, time_dimension_size):
    LWu = calculate_LWu(Ts[n-1])

    Ts[n] = Ts[n-1] + dt/C_SOIL * ( (1-albedo) * SWd[n-1] + LWd - LWu - H[n-1] - LE[n-1] - G )

### Put Ts into xarray
Ts = xr.DataArray(Ts, dims='time', coords={'time': datetime_UTC_2025},
                 attrs={'units': 'K',
                        'long_name': 'Surface temperature'},)
